Reckless is an open source competitive chess engine written in Rust. It communicates using the Universal Chess Interface (UCI) protocol and is designed to be used with a UCI-compatible GUI. Since its first release in May 2023, Reckless has climbed from a rating of ~2005 to over 3800, placing among the top two engines across major rating lists.

Reckless competes regularly in major tournaments including the [Chess.com Computer Chess Championship (CCC)](https://www.chess.com/computer-chess-championship) and the [Top Chess Engine Championship (TCEC)](https://tcec-chess.com).

## Key features

<CardGroup cols={2}>
  <Card title="UCI protocol" icon="plug">
    Full UCI support, compatible with any standard chess GUI or testing framework.
  </Card>
  <Card title="Multi-threaded search" icon="microchip">
    Scales across up to 512 threads using shared-memory parallel search.
  </Card>
  <Card title="NNUE evaluation" icon="brain">
    Neural network evaluation trained with [Bullet](https://github.com/jw1912/bullet) for accurate positional judgment.
  </Card>
  <Card title="Syzygy tablebases" icon="database">
    Endgame tablebase support via the Fathom library for perfect endgame play.
  </Card>
  <Card title="Chess960 support" icon="shuffle">
    Fischer Random Chess support via the `UCI_Chess960` option.
  </Card>
  <Card title="Profile-guided optimization" icon="gauge">
    PGO builds available for maximum performance on your specific hardware.
  </Card>
</CardGroup>
