Reckless is not a standalone chess program. It is a chess engine — a back-end process that calculates moves. To play against it or use it for analysis, you need to connect it to a UCI-compatible graphical user interface (GUI).

<Note>
  UCI (Universal Chess Interface) is a standard communication protocol that lets chess GUIs talk to engines. The GUI sends commands like `go` and `position`, and the engine responds with its best move. You don't need to understand UCI in detail to use Reckless — the GUI handles it automatically.
</Note>

## Compatible GUIs

Any UCI-compatible chess GUI will work with Reckless. The following are commonly used options:

| GUI | Platform | Notes |
| --- | -------- | ----- |
| [Cute Chess](https://github.com/cutechess/cutechess) | Windows, macOS, Linux | Widely used for engine tournaments and testing |
| [En Croissant](https://encroissant.org) | Windows, macOS, Linux | Modern interface with built-in engine management |
| [Nibbler](https://github.com/rooklift/nibbler) | Windows, macOS, Linux | Designed for analysis with real-time engine output |

## Connect Reckless to a GUI

The exact steps vary by GUI, but the general process is the same for all of them.

<Steps>
  <Step title="Install a UCI-compatible GUI">
    Download and install one of the GUIs listed above. Follow its installation instructions before proceeding.
  </Step>

  <Step title="Get the Reckless binary">
    If you haven't already, download a precompiled binary from the [GitHub Releases page](https://github.com/codedeliveryservice/Reckless/releases) or [build from source](/installation).

    On Linux and macOS, make the binary executable:

    ```bash
    chmod +x reckless-avx2
    ```
  </Step>

  <Step title="Add Reckless as an engine in your GUI">
    Open your GUI's engine management settings (often found under **Engines** or **Preferences**) and add a new engine. When prompted, point the GUI to the Reckless binary on your filesystem.

    The GUI will automatically run the binary and communicate with it over UCI.
  </Step>

  <Step title="Start a game or analysis session">
    Select Reckless as your engine and start a new game or load a position for analysis. The GUI will handle all communication with the engine.
  </Step>
</Steps>

## Verify the engine works

If you want to confirm that the binary is working correctly before connecting it to a GUI, you can interact with it directly in your terminal using the UCI protocol.

Run the binary and type `uci`:

```text
uci
```

Reckless will respond with its identity and the list of supported options:

```text
id name Reckless v0.9.0
id author Arseniy Surkov, Shahin M. Shahin, and Styx
option name Hash type spin default 16 min 1 max 262144
option name Threads type spin default 1 min 1 max 512
option name MoveOverhead type spin default 100 min 0 max 2000
option name Minimal type check default false
option name Clear Hash type button
option name UCI_Chess960 type check default false
option name MultiPV type spin default 1 min 1 max 218
option name SyzygyPath type string default
uciok
```

If you see `uciok` at the end, the engine is working. Type `quit` to exit.
