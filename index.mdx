Reckless is a high-performance chess engine that communicates via the Universal Chess Interface (UCI) protocol. It combines a hand-crafted NNUE neural network with advanced alpha-beta search techniques, multi-threaded parallel search, and Syzygy endgame tablebase support.

Reckless consistently performs among the top engines in major tournaments including the [Chess.com Computer Chess Championship (CCC)](https://www.chess.com/computer-chess-championship) and [Top Chess Engine Championship (TCEC)](https://tcec-chess.com).

## Key features

<CardGroup cols={2}>
  <Card title="NNUE Evaluation" icon="network-wired">
    3-layer neural network with AVX-512, AVX2, and NEON SIMD acceleration for fast, accurate position evaluation
  </Card>
  <Card title="Advanced Search" icon="magnifying-glass">
    Alpha-beta search with PVS, aspiration windows, null move pruning, singular extensions, and LMR
  </Card>
  <Card title="Multi-threaded" icon="microchip">
    NUMA-aware thread pool supporting up to 512 parallel search threads
  </Card>
  <Card title="Syzygy Tablebases" icon="table">
    Endgame tablebase support via the Fathom library for perfect play in late endgames
  </Card>
  <Card title="Chess960 Support" icon="chess-board">
    Full Fischer Random Chess (Chess960) support via the UCI_Chess960 option
  </Card>
  <Card title="Profile-Guided Optimization" icon="chart-line">
    PGO builds for maximum performance using real game data to guide compiler optimizations
  </Card>
</CardGroup>

## Getting started

<Steps>
  <Step title="Download or build Reckless">
    Grab a precompiled binary from the [GitHub Releases page](https://github.com/codedeliveryservice/Reckless/releases) or [build from source](/building/from-source).
  </Step>
  <Step title="Install a UCI-compatible GUI">
    Reckless works with any UCI GUI. Popular choices include [Cute Chess](https://github.com/cutechess/cutechess), [En Croissant](https://encroissant.org), and [Nibbler](https://github.com/rooklift/nibbler).
  </Step>
  <Step title="Add the engine to your GUI">
    Point your GUI to the Reckless binary. The engine will be detected automatically as a UCI engine.
  </Step>
  <Step title="Configure options">
    Adjust [UCI options](/uci/options) such as `Hash`, `Threads`, and optionally `SyzygyPath` for tablebase support.
  </Step>
</Steps>
